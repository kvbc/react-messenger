# react-messenger

## Installing

- Run `install.bat`

or

1. Run `npm run install-link` in `shared/`
2. Run `npm run install-link` in `frontend/` and `backend/`

## Running / Development

1. Run `dev.bat` or run `npm run dev` in all `backend/`, `frontend/` and `shared/`
2. Open up `http://localhost:3000`
