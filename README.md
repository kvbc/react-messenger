# react-messenger

## Installing

1. Run `install.bat`

or

1. Run `npm run install-link` in `shared/`
2. Run `npm run install-link` in `frontend/` and `backend/`

## Running / Development

1. Run `dev.bat /ob`

or

1. Run `npm run dev` in `backend/`, `frontend/` and `shared/`
2. Open up `http://localhost:3000`
